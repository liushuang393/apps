1. エグゼクティブサマリー

ビジョン

「すべての会話を企業の競争力に変える」
AI技術を活用し、営業面談・カスタマーサポート・社内会議を分析することで、企業の営業成果向上と組織学習を加速させるプラットフォーム

市場機会

日本の中堅企業における営業デジタル化の需要は急速に拡大しています。特に：

  • 営業力の可視化：営業スキル評価が属人的で、教育効果が測定できない
  • 顧客ニーズの把握遅延：会議録音後の分析に数日かかり、営業対応が遅れる
  • 組織学習の欠如：優秀営業のノウハウが共有されず、チーム全体の成長が阻害される

2. 製品概要と価値提案

ConverSightとは

ConverSight は、Teams、Zoom、Google Meet などのビデオ会議プラットフォームに統合される、企業向けAI会議分析システムです。営業面談やカスタマーサポート会議をリアルタイムで分析し、経営陣・営業マネージャー・営業担当者に対して、即座に行動可能な洞察を提供します。

コンセプト

「AIが会話を聞き、理解し、洞察を提供する」
AI listens, understands, and empowers your meetings.

差別化要因

⚡

リアルタイム性

会議中の2～3分ごとに営業者へ提示。事後分析ではなく、その場での行動変容が可能

🎯

営業特化

購買意欲、異議対応、セールステクニック評価など、営業特有のKPIに特化

🌐

多言語対応

日本語、英語、中国語に対応。グローバル企業や国際営業にも対応可能

📊

定量評価

営業スキルを「レーダーチャート」で可視化。CRM・BI連携で経営判断に活用可能

3. コア機能

機能① 実時間転録と記録（Real-time Transcription）

Teams、Zoom、Google Meet などのプラットフォームから音声を自動取得し、リアルタイムで日本語、英語、中国語に転録します。転録結果は完全検索可能な形式で保存され、会議アーカイブとしてチーム全体で共有できます。

  • 自動音声認識（ASR）：95%以上の精度で日本語転録
  • 話者識別機能：複数参加者を自動区別
  • 検索可能なアーカイブ：5年間保存、タグ・カテゴリ分類
  • エクスポート機能：PDF / Word / Excel / JSON形式対応

機能② 会議中リアルタイム洞察（In-meeting Insight）

営業者のパフォーマンスと顧客の反応を2～3分ごとに分析し、リアルタイム提示します。営業者はモニター画面でアラート受け取り、その場で戦術を調整できます。

分析項目 表示形式 営業者への推奨アクション例
顧客の購買関心度 🟢 (高) / 🟡 (中) / 🔴 (低) 🟢の場合：「詳細説明へ進む準備完了」
顧客の感情・トーン スコア表示 (0-100) 低下時：「倾聴モード、相手のニーズ引き出し」
主要な関心キーワード リアルタイム抽出 「価格」言及：「価値説明に注力」
異議・懸念の検出 ⚠️ アラート 「ここで対応策を提示して解決」

機能③ 会後総結分析レポート（Post-meeting Analysis）

会議終了後、自動的に包括的な分析レポートが生成されます。顧客分析、営業分析、フォローアップ提案を含みます。

📊 顧客分析パート

  • 購買度合スコア（0-100）
  • 検討フェーズの推定（認知 / 興味 / 比較検討 / 決定段階）
  • 主要関心領域（機能 / 価格 / 導入リスク等）
  • 競合品への言及有無と内容
  • 次のアクション推奨日程

🎯 営業分析パート

  • 営業活動スコア（説明力 / 傾聴力 / 異議対応力 / クロージング力）
  • 発言時間バランス（営業者:顧客の理想比は 40:60）
  • 質問の質と深さ（オープン質問 vs クローズ質問の比率）
  • 建設的フレームワーク使用度
  • 個別改善提案

機能④ チーム/組織レベルの可視化ダッシュボード

マネージャーや経営陣向けに、営業チーム全体のパフォーマンスを一元管理するダッシュボードを提供します。

  • 営業者ランキング（成約率、顧客満足度）
  • チーム営業スキルレーダーチャート
  • 業界別・顧客セグメント別の成約率分析
  • コーチング対象者の自動抽出
  • ベストプラクティスの可視化（成功事例の共有）

オプション機能

  • 音声トーン分析：話速、抑揚、感情的起伏を検出
  • 動画フレーム分析：非言語コミュニケーション（ジェスチャー、アイコンタクト）の分析
  • 競合インテリジェンス：競合品への言及パターン分類
  • 多言語リアルタイム翻訳：グローバル会議での同時翻訳表示

4. 技術アーキテクチャ

システム構成

  • フロントエンド：Next.js v16.0.1 + TypeScript + Tailwind CSS + shadcn/ui（高度にカスタマイズ可能なダッシュボード）
  • バックエンド：Node.js / Python FastAPI（処理エンジン）
  • AI/ML エンジン（OpenAI）：
    • ASR（音声認識）：gpt-4o-transcribe / gpt-4o-mini-transcribe（日本語対応、WER改善、背景ノイズ対応）
    • 話者識別：gpt-4o-transcribe-diarize（Diarization機能付き）
    • 感情分析 / 意図抽出：GPT-4o（テキスト分析）+ o4-mini（高速推論）
    • テキスト音声化：gpt-4o-mini-tts（自然な日本語TTS、Steerability対応）
  • リアルタイム通信：WebSocket + OpenAI Realtime API（低レイテンシ音声対話）
  • データベース：PostgreSQL（メタデータ）+ Vector DB（埋め込みベクトル、GPT-4o embedding）
  • インテグレーション：Teams API / Zoom API / SalesForce API / HubSpot API

セキュリティモデル

  • エンドツーエンド暗号化（音声・テキスト）
  • API キー / OAuth 2.0 認証
  • ロールベースアクセス制御（RBAC）
  • 監査ログ完全記録
  • GDPR / 個人情報保護法準拠

6. 市場分析と競争優位性

日本市場の背景

  • 営業DX化の加速：コロナ禍によるオンライン面談の常態化で、オンライン営業分析ツールへの需要が急速に拡大
  • 世代交代による営業ノウハウの損失課題：ベテラン営業の属人的スキルが形式知化されず、組織全体の育成効率が低下
  • 働き方改革による労働時間制約：営業事務作業の自動化ニーズが高まっている

競争環境

既存プレイヤー

  • Otter.ai / Fireflies.ai：会議転録が主機能、営業特化の分析機能が弱い
  • Microsoft Copilot（Meetings）：Teams標準機能、カスタマイズ性が低い
  • 日本ベンダー：営業支援SFA（SalesForce、Hubspot）の音声分析機能追加中だが、リアルタイム性に欠ける

ConverSightの競争優位性

🎯 営業特化設計
一般的な会議分析ツール（Otter.ai）と異なり、営業プロセスの各段階（初接触→提案→交渉→クロージング）に最適化された分析ロジックを搭載
⚡ リアルタイム性
会議中の2～3分ごとにアクショナブルな提示、事後分析のみの他社と差別化
🌐 多言語・日本語特化
日本語の複雑な敬語・文脈を正確に分析し、日本市場ニーズに完全対応
📊 定量的スキル評価
営業能力を「レーダーチャート」で可視化し、CRM・HRシステムと連携した評価体制を構築可能

8. セキュリティ＆コンプライアンス

データセキュリティ方針

  • 暗号化：AES-256 で全転送データを暗号化、保存時も暗号化
  • アクセス制御：ロールベースアクセス制御（RBAC）、多要素認証（MFA）
  • 監査ログ：全アクセス・操作ログを5年間保存、改ざん防止
  • 定期セキュリティ監査：年1回外部セキュリティ監査実施、脆弱性スキャン月1回

法令準拠

  • 個人情報保護法（APPI）：日本国内データの厳格管理
  • GDPR：EUデータの適切な処理（EU企業向け）
  • 金融業務：信託銀行向けオンプレミス版オプション

プライバシー保護

  • 音声録音は参加者全員の同意を取得
  • 分析対象外にするキーワード設定可能（例：機密情報、個人情報）
  • レポート出力時の秘匿設定（例：顧客名マスキング）

「AI技術を営業の力に変える」
